# Reviews and References

This subrepository stores public reference-style feedback submitted by past coworkers, managers, employees, employers, customers, and other professional contacts for Casey Lide.

## Purpose

The purpose of this folder is to preserve submitted feedback as simple public text files inside the `caseylide/public-portfolio` repository so that:

- feedback can be transparently stored in version control
- submissions can be displayed on the public portfolio website
- each submission remains easy to review, audit, and reference

## Storage Convention

Each review submission is stored as its own plain-text file directly inside this folder.

The file naming convention is:

- `<email>.1.txt`
- `<email>.2.txt`
- `<email>.3.txt`

Examples:

- `jane@doe.com.1.txt`
- `jane@doe.com.2.txt`
- `john.smith@company.com.1.txt`

This allows multiple submissions from the same email address while preserving submission order.

## Expected Submission Contents

Each submitted file may contain the following information:

- first name
- last name
- email address
- feedback text
- phone number (optional)
- current company (optional)
- current role (optional)
- submission timestamp

## Publication Note

Only this `README.md` file is intended to establish and document the `reviews-and-references` subrepository structure. Review entries are expected to be added programmatically by the public website as individual text files in this folder.
